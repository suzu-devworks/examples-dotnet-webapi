# examples-dotnet-webapi

## ASP.NET Core WebAPI 

### The way to the present

```shell
git clone https://github.com/suzu-devworks/examples-dotnet-webapi.git

dotnet new sln -o .
dotnet new webapi -o ExamplesWebapi
dotnet add sln  ./ExamplesWebapi/ExamplesWebapi.csproj

```
